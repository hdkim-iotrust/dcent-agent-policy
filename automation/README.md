매일 오전 10시, 본인 구글 계정으로 실행되어:
1. 검수 추적 시트를 읽고
2. **AI 답변(H)이 비어 있는 행 제외** (매크로 처리, 문의 미비 직접답변 등 → 학습할 초안 없음)

4. 변경유형(RULE/FACT)을 가볍게 제안하고
5. 변경이력 시트에 **"검토 대기"**로 추가합니다. (이미 있는 건 중복 추가 안 함)

## 추적 시트 열 매핑 (2026-07 기준)
`weekly_review.gs`의 `COL_*` 값이 아래 열과 일치해야 합니다. 시트 열이 또 바뀌면 이 값만 수정하세요.
- 티켓번호 = B(1), AI agent 답변 내용 = H(7), 수정 = O(14), 수정 근거 및 사유 = P(15)
- (참고: A#, C답변날짜, D답변시간, E상태, F제목, G문의내용, I~N 6원칙 체크, Q보완)

## 설정 / 갱신 (비개발자용)

1. **변경이력 시트** 열기 → 상단 메뉴 **확장 프로그램 → Apps Script**
2. 편집기 코드를 `automation/weekly_review.gs` 최신 내용으로 **전체 교체** → 저장(💾)
3. 함수 목록에서 **dailyReview** 선택 → **▶ 실행**
   - 처음 실행 시 권한 요청 → 본인(hdkim@iotrust.kr) 계정으로 **허용**
   - (변경이력 시트에 '검토 대기' 행이 추가되면 정상)
4. 좌측 **시계 아이콘(트리거)** → 기존 트리거가 있으면 **편집**, 없으면 **트리거 추가**
   - 실행할 함수: `dailyReview`  (기존 weeklyReview 트리거는 편집으로 이 함수로 바꾸거나 삭제)
   - 이벤트 소스: **시간 기반**
   - 트리거 유형: **일 단위 타이머** → **오전 10시~11시 사이**
   - 저장
5. 끝. 이후 매일 오전 10시 자동 실행됩니다.

> 참고: 본인 계정이 두 시트 모두 접근 권한이 있어야 합니다(추적=보기 이상, 변경이력=편집).
> ※ 함수명을 dailyReview로 바꿨으므로, 기존 weeklyReview 트리거를 그대로 두면 "함수 없음" 오류가 납니다. 반드시 편집 또는 삭제하세요.

## 제외/포함 규칙
- 포함(학습 후보): "수정"(O) 칸에 내용이 있는 행 = AI 초안을 사람이 고친 건
- 제외:
  - "수정"(O) 칸이 비어 있는 행(초안 그대로 나감)
  - **"AI agent 답변 내용"(H)이 비어 있는 행** = 에이전트 초안이 없는 경우(매크로 처리, 문의 미비로 직접 답변 등) → 학습할 초안이 없으므로 제외
